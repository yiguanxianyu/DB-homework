# 数据库概论 作业一 学生数据管理系统

- 这只是一次课程作业

- 运行前请根据`requirements.txt`安装所需的运行库。或者直接在命令行输入：

```console
$pip3 install pycryptodome
```

## 特点

1. 支持自动识别学号和姓名
2. 支持重名学生的管理
3. 数据文件加密存储，安全性高
4. 采用 HashMap 存取数据，时间复杂度仅为 O(1)

## 限制

1. 学号仅限纯数字，姓名不得为纯数字
2. 需要一次性将所有数据全部读入内存中
3. 程序若异常退出，所做的所有更改将无法保存

## Q&A

### 本项目采用哪种编码方式?如何保证数据安全？

本项目采用二进制编码，将学生数据的文本格式通过`aes-256-cfb`加密后以二进制形式存储在`students.dat`文件中，以防止外部软件对信息的盗取。即使第三者拥有了本项目的全部源代码，在未知密码的情况下仍将不能获得明文数据。你可以自定义默认的密码，但应该为

### 本项目采用哪种数据结构？

本项目中，学生信息表和学生均以对象的形式存储，而学生的具体信息以字典的形式存储。其中学生的成绩栏以一个内嵌的字典实现。其中主要使用的数据结构是哈希表 (HashMap)，以学生的学号为键 (Key)，学生对象的实例为值 (Value)，实现对学生数据的存取，并能达到 O(1) 的时间复杂度。HashMap 的实现借助了python自带的容器字典 (Dict)。具体结构如下图所示。
